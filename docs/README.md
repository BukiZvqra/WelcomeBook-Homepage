# docs/

Технически документи за WelcomeBook Agency monorepo.

## Какво има тук

- `architecture-v1.4.md` — пълна архитектурна спецификация (single source of truth за build процеса)
- `brand-spec-v1.0.md` — brand identity (colors, typography, voice, copy rules)

## Кога чета тези файлове

Преди:
- Build на нов client website или portal
- Промяна в shared template или backend
- Setup на нов клиент

При промяна в тези документи:
- Architecture: Update version number + changelog при всяка значителна промяна
- Brand spec: Update version при visual identity промени

## Source of truth

Тези файлове живеят на 2 места едновременно:
1. Тук в repo-то (за Claude Code сесии)
2. В Project Knowledge на claude.ai (за чат сесии)

При update на единия — обнови и другия.
